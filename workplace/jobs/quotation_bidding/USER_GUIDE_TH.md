# คู่มือใช้งานระบบ Quotation Bidding

คู่มือนี้เป็นเวอร์ชันสั้นสำหรับผู้ใช้งานทั่วไป ครอบคลุมการเข้าสู่ระบบ การสมัครสมาชิก การรีเซ็ตรหัสผ่าน และวิธีใช้งานพื้นฐานของ `Admin` และ `Vendor/Supplier`

## 1. การเข้าสู่ระบบ

1. กรอก `Username`
2. กรอก `Password`
3. กด `Sign In`

ถ้าเข้าสู่ระบบสำเร็จ ระบบจะพาไปยังหน้าทำงานตามสิทธิ์ของผู้ใช้โดยอัตโนมัติ

## 2. การสมัครสมาชิก

สำหรับผู้ขายหรือ Supplier ที่ยังไม่มีบัญชี

1. ที่หน้า Login กด `Create an account`
2. กรอกข้อมูลให้ครบ
   - `Contact Name`
   - `Supplier Name`
   - `Email`
   - `Username`
   - `Password`
3. กด `Create Account`
4. เมื่อสมัครสำเร็จ กด `Back to Login`
5. เข้าสู่ระบบด้วยบัญชีใหม่

หมายเหตุ:
ใช้อีเมลจริงที่สามารถรับ OTP และอีเมลแจ้งเตือนได้

## 3. การรีเซ็ตรหัสผ่าน

1. ที่หน้า Login กด `Forgot password?`
2. กรอกอีเมลที่ลงทะเบียนไว้
3. กด `Send OTP`
4. นำรหัส OTP จากอีเมลมากรอก
5. กด `Verify OTP`
6. กรอกรหัสผ่านใหม่
7. กด `Save New Password`
8. กลับไป Login ใหม่

หากไม่พบอีเมล OTP ให้ตรวจสอบโฟลเดอร์ Spam/Junk Mail

## 4. การใช้งานสำหรับ Admin

Admin ใช้สำหรับจัดการงานและตรวจสอบใบเสนอราคา

### งานที่ทำได้

- สร้าง Work Order ใหม่
- แก้ไข Work Order
- ดูจำนวน QT ที่เข้ามาในแต่ละงาน
- กด `Compare` เพื่อเปรียบเทียบ QT
- กรอกราคา `THB` และบันทึกหมายเหตุ
- เลือก QT ที่ต้องการใช้งาน
- จัดการผู้ใช้งานผ่านเมนู `Manage Users`

### เมนูสำคัญ

- `Create Work Order`
- `Manage Users`
- `Refresh Data`
- `Log Out`

### การแจ้งเตือน

- ระบบมีการแจ้งเตือนเมื่อมี QT ใหม่
- จะมี dot หรือ badge แสดงรายการที่มีอัปเดต
- สามารถกดเข้าไปดูรายละเอียดงานนั้นได้ทันที

## 5. การใช้งานสำหรับ Vendor / Supplier

Vendor ใช้สำหรับดูงานที่เปิดรับราคาและส่ง QT

### งานที่ทำได้

- ดูรายการ `Available Work Orders`
- ค้นหาและ sort งาน
- กด `Add QT` เพื่อส่งใบเสนอราคาใหม่
- กด `Manage QT` เพื่อดูหรือแก้ไข QT เดิม
- ดูรายการ QT ของตัวเองผ่าน `My Quotations`

### ขั้นตอนส่ง QT

1. เลือกงานที่ต้องการ
2. กด `Add QT`
3. กรอกข้อมูลใบเสนอราคา
4. แนบรูปสินค้าในส่วน `Product Images` ถ้ามี
5. กด `Submit Quotation`

### เมนูสำคัญ

- `My Quotations`
- `Refresh Data`
- `Log Out`

## 6. คำแนะนำการใช้งาน

- หากต้องการข้อมูลล่าสุด ให้กด `Refresh Data`
- ก่อนกดบันทึก ควรรอให้อัปโหลดไฟล์หรือรูปเสร็จสมบูรณ์
- หากระบบเด้งออกหรือขึ้นว่า session หมดอายุ ให้เข้าสู่ระบบใหม่

## 7. ปัญหาที่พบบ่อย

### เข้าระบบไม่ได้

- ตรวจสอบ Username และ Password
- หากลืมรหัสผ่าน ให้ใช้ `Forgot password?`

### สมัครสมาชิกไม่ได้

- ตรวจสอบว่า Email หรือ Username ไม่ซ้ำ
- ตรวจสอบว่ากรอกข้อมูลครบ

### ไม่ได้รับ OTP

- ตรวจสอบ Spam/Junk Mail
- ตรวจสอบว่าใช้อีเมลที่ลงทะเบียนไว้จริง

### บันทึกงานหรือ QT ไม่ได้

- ตรวจสอบว่ากรอกข้อมูลครบ
- ตรวจสอบว่าไฟล์แนบอัปโหลดเสร็จแล้ว
